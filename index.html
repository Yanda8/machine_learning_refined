The set of posts, cut into short series, use careful writing and interactive coding widgets to provide an intuitive and playful way to learn about core concepts in machine learning.

Every post is a Python Jupyter notebook, prettied up for the web, that you can download and run on your own machine by pulling our GitHub repo. These posts were created as rough draft material for the second edition of the textbook Machine Learning Refined published by Cambridge University Press.
